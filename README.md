Opinionated Babel preset

## Why?
I want to reuse my openionated babel setting most of the time.

## Details
[In here](index.js)
## How to use

### Install
- `npm install --save-dev @comeon/babel-preset-comeon`

### Add in .babelrc
```json
{
  "presets": ["@comeon/comeon"]
}
```
